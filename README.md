**MaryJaneCoin (MARYJ)** is a privacy-focused Proof-of-Stake cryptocurrency with a 1 billion MARYJ hard cap supply. It features a bridge escrow model allowing users to wrap MARYJ onto Solana as a pump.fun SPL token, enabling DEX trading and smart contract integration. Staking earns 0% APR — rewards come from transaction fees only (0.420 MARYJ per TX). Privacy is layered and additive: network privacy via Dandelion++, wallet-level CoinJoin, and Mimblewimble Extension Blocks (MWEB) planned for protocol-level private spending. Transparent staking coexists with private spending in the MWEB model.
